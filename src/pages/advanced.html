The hostname is a user-friendly label\n assigned to the router to make it easier to identify and access on a network.\n
\n The third octet instructs the router which IP address to use. Valid values range\n from 0 to 255 and are used in the third position within the IP address. This\n changes both the IP of the router and the IP of all connected clients.\n
\n\n
\n
\n
\n \n \n
\n
\n \n \n
\n\n
\n The transmission power determines how strong the AP (Access Point) network is.\n Higher values mean greater range.\n
\n The bandwidth affects the speed of the connection between the router and\n clients. The higher the value, the slower it\n becomes. Higher values also imply greater stability, especially in the\n case of frequent network interruptions.\n\n

This overrides the DNS server of the\n uplink network. Changing this results in more privacy (Cloudflare) or the\n ability to block advertisements(AdGuard).\n Use this with caution if you have to register/login to the uplink network. This\n can lead to unreachable sites. After login or registration this can be\n reactivated. If you define an invalid custom DNS\n server, the value will be set back to default one.

MAC override

Your current MAC address is: %s\n
\n
\n
\n
\n
This overrides the MAC address of the\n device. Changing\n this can be used for networks with MAC limitations. For example, when the\n network allows only a limited number of devices per access for specific MACs.\n The value will be\n resetted to the default, if you define an invalid address. To\n bypass time by device limits in some WiFis (e.g. 30min free per device and day)\n you can select "Random". This will change the last part of the default MAC on\n every reboot.\n
\n
\n

Netmask override

Your current netmask is: %s\n
\n
\n \n
\n
\n
\n
\n
\n
This overrides the netmask of the\n device and can be\n used if upstream network uses a non class C mask. The setting can be used for\n communication with\n clients connected to the upstream network. This changes both the IP of the\n router and the IP of all connected clients.
